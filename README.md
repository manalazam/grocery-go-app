# GroceryGo - Smart Grocery Management App

## Project Description

GroceryGo is a Flutter-based grocery management application that helps users organize, track, and manage their shopping items efficiently.

The application allows users to create grocery lists, categorize products, search for items, track purchased products, and view shopping insights through a clean and user-friendly interface.

---

### User Management

* User Login
* User Registration

### Grocery Management

* Add grocery items
* Edit grocery items
* Delete grocery items
* Mark items as purchased

### Search Functionality

* Search products instantly
* Quick product lookup

### Insights Dashboard

* View total items

## REST API Implementation

This project implements REST APIs for communication between the Flutter frontend and Node.js backend.

### API Operations

* GET → Fetch grocery items
* POST → Add new grocery items
* PUT → Update grocery items
* DELETE → Remove grocery items

REST APIs helped establish efficient communication between the mobile application and the database.
